Efectos de JQUERY

Los efectos de jQuery permiten cambiar propiedades de estilo, posición, visibilidad, tamaño, de los elementos de manera animada y controlada. Estos efectos pueden aplicarse tanto al cargar la página como en respuesta a eventos como clics, desplazamientos, etc. Estos efectos se logran utilizando los métodos y funciones proporcionados por jQuery, como fadeIn(), fadeOut(), slideToggle(), animate().

jQuery Ocultar/Mostrar

  • jQuery hide()

    La función hide() en jQuery se utiliza para ocultar los elementos seleccionados en una página web. Puede ser llamada en cualquier objeto jQuery, que generalmente es un conjunto de elementos HTML seleccionados mediante un selector CSS u otros métodos de jQuery.

    Código

                                        <script>
                                      $(document).ready(function(){
                                        $("p").click(function(){
                                          $(this).hide();
                                        });
                                      });
                                    </script>

                                    <p>If you click on me, I will disappear.</p>
                                    <p>Click me away!</p>
                                    <p>Click me too!</p>

    Funcionamiento

    If you click on me, I will disappear.

    Click me away!

    Click me too!

  • jQuery hide() and show()

    La función hide() en jQuery se utiliza para ocultar los elementos seleccionados, mientras que la función show() se utiliza para mostrar los elementos ocultos previamente.

    Código

                                            <script>
                                        $(document).ready(function(){
                                          $("#hide").click(function(){
                                            $("p").hide();
                                          });
                                          $("#show").click(function(){
                                            $("p").show();
                                          });
                                        });
                                        </script>
                                        <p>If you click on the "Hide" button, I will disappear.</p>
                                        <button id="hide">Hide</button>
                                        <button id="show">Show</button>

    Funcionamiento

    If you click on the "Hide" button, I will disappear.


  • jQuery toggle()

    La función toggle() en jQuery se utiliza para alternar la visibilidad de los elementos seleccionados. Si un elemento está oculto, toggle() lo mostrará, y si está visible, lo ocultará. 

                                            <script>
                                            $(document).ready(function(){
                                              $("button").click(function(){
                                                $("p").toggle();
                                              });
                                            });
                                        </script>
                                        <button>Toggle between hiding and showing the paragraphs</button>
                                            
                                        <p>This is a paragraph with little content.</p>
                                        <p>This is another small paragraph.</p>

    Funcionamiento

    This is a paragraph with little content.

    This is another small paragraph.

jQuery Fade

  • jQuery fadeIn()

    La función fadeIn() en jQuery se utiliza para mostrar gradualmente los elementos seleccionados, aplicando una transición de opacidad. Inicialmente, los elementos están ocultos y luego se vuelven gradualmente visibles.

    Código

                                            <script>
                                        $(document).ready(function(){
                                            $("button").click(function(){
                                                $("#div1").fadeIn();
                                                $("#div2").fadeIn("slow");
                                                $("#div3").fadeIn(3000);
                                            });
                                        });
                                        </script>

                                        <p>Demonstrate fadeIn() with different parameters.</p>
                                        
                                        <button>Click to fade in boxes</button><br>
                                        <br>
                                        <div id="div1" style="width:80px;height:80px;display:none;background-color:red;"></div>
                                        <br>
                                        <div id="div2" style="width:80px;height:80px;display:none;background-color:green;"></div>
                                        <br>
                                        <div id="div3" style="width:80px;height:80px;display:none;background-color:blue;"></div>

    Demonstrate fadeIn() with different parameters.





  • jQuery fadeOut()

    La función fadeOut() en jQuery se utiliza para ocultar gradualmente los elementos seleccionados, aplicando una transición de opacidad. Los elementos se vuelven gradualmente transparentes hasta que estén completamente ocultos.

    Código

                                            <script>
                                        $(document).ready(function(){
                                          $("button").click(function(){
                                            $("#div1").fadeOut();
                                            $("#div2").fadeOut("slow");
                                            $("#div3").fadeOut(3000);
                                          });
                                        });
                                        </script>
                                        
                                        <p>Demonstrate fadeOut() with different parameters.</p>
                                        
                                        <button>Click to fade out boxes</button><br>
                                        <br>
                                        
                                        <div id="div1" style="width:80px;height:80px;background-color:red;"></div>
                                        <br>
                                        <div id="div2" style="width:80px;height:80px;background-color:green;"></div>
                                        <br>
                                        <div id="div3" style="width:80px;height:80px;background-color:blue;"></div>

    Demonstrate fadeOut() with different parameters.





  • jQuery fadeToggle()

    La función fadeToggle() en jQuery se utiliza para alternar la visibilidad de los elementos seleccionados mediante una animación de desvanecimiento. Si los elementos están visibles, fadeToggle() los ocultará gradualmente; si están ocultos, fadeToggle() los mostrará gradualmente.

    Código

                                            <script>
                                        $(document).ready(function(){
                                          $("button").click(function(){
                                            $("#div1").fadeOut();
                                            $("#div2").fadeOut("slow");
                                            $("#div3").fadeOut(3000);
                                          });
                                        });
                                        </script>
                                        
                                        <p>Demonstrate fadeOut() with different parameters.</p>
                                        
                                        <button>Click to fade out boxes</button><br><br>
                                        
                                        <div id="div1" style="width:80px;height:80px;background-color:red;"></div>
                                        <br>
                                        <div id="div2" style="width:80px;height:80px;background-color:green;"></div>
                                        <br>
                                        <div id="div3" style="width:80px;height:80px;background-color:blue;"></div>

    Demonstrate fadeToggle() with different speed parameters.





  • jQuery fadeTo()

    La función fadeTo() en jQuery se utiliza para cambiar gradualmente la opacidad de los elementos seleccionados a un valor específico. Permite ajustar la opacidad de los elementos de forma personalizada.

    Código

                                            <script>
                                        $(document).ready(function(){
                                          $("button.fadeto").click(function(){
                                            $("#div10").fadeTo("slow", 0.15);
                                            $("#div11").fadeTo("slow", 0.4);
                                            $("#div12").fadeTo("slow", 0.7);
                                          });
                                        });
                                        </script>
                                        
                                        <p>Demonstrate fadeTo() with different parameters.</p>
                                        
                                        <button class="fadeto">Click to fade boxes</button><br><br>
                                        
                                        <div id="div10" style="width:80px;height:80px;background-color:red;"></div>
                                        <br>
                                        <div id="div11" style="width:80px;height:80px;background-color:green;"></div>
                                        <br>
                                        <div id="div12" style="width:80px;height:80px;background-color:blue;"></div>

    Demonstrate fadeTo() with different parameters.

  • jQuery stop() animation ( with parameters)

    Código

                                            <script> 
                                        $(document).ready(function(){
                                          $("#start").click(function(){
                                            $("div").animate({left: '100px'}, 5000);
                                            $("div").animate({fontSize: '3em'}, 5000);
                                          });
                                          
                                          $("#stop").click(function(){
                                            $("div").stop();
                                          });
                                        
                                          $("#stop2").click(function(){
                                            $("div").stop(true);
                                          });
                                        
                                          $("#stop3").click(function(){
                                            $("div").stop(true, true);
                                          }); 
                                        });
                                        </script> 
                                        </head>
                                        <body>
                                        
                                        <button id="start">Start</button>
                                        <button id="stop">Stop</button>
                                        <button id="stop2">Stop all</button>
                                        <button id="stop3">Stop but finish</button>
                                        
                                        <p>The "Start" button starts the animation.</p>
                                        <p>The "Stop" button stops the current active animation, 
                                        but allows the queued animations to be performed afterwards.</p>
                                        <p>The "Stop all" button stops the current active animation and clears the 
                                        animation queue; so all animations on the element is stopped.</p>
                                        <p>The "Stop but finish" rushes through the current active animation, 
                                        then it stops.</p> 
                                        
                                        <div style="background:#98bf21;height:100px;
                                        width:200px;position:absolute;">HELLO</div>

    The "Start" button starts the animation.

    The "Stop" button stops the current active animation, but allows the queued animations to be performed afterwards.

    The "Stop all" button stops the current active animation and clears the animation queue; so all animations on the element is stopped.

    The "Stop but finish" rushes through the current active animation, then it stops.
